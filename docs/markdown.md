
### Center Text

```
<div align="center">
# This is gonna be centered!
</div>
```

### Center Image

```
<div align="center">
<img src="path/to/image.jpg" width="250"/>
</div>
```

### Colorize Text

```
$`\textcolor{green}{\text{green}}`$
$`\textcolor{orange}{\text{orange}}`$
$`\textcolor{red}{\text{red}}`$
```
